# 💎 BrowserForge Investor Portfolio

A premium, high-converting investor landing page and data dashboard showing the security and scale of the BrowserForge network.

## 🚀 Key Features

- **Architecture Transparency:** Detailed breakdown of the Build Farm infrastructure.
- **Security Audit:** Live metrics showing security scores across the global fleet.
- **Secret Access:** Verified tunnel links for direct production monitoring.
- **MDX-Powered:** High-performance content management for professional presentations.

## 🏗️ Deployment

This portfolio is built with Next.js and Once UI, optimized for instant performance on Vercel.

```bash
npm install
npm run dev
```

## 🛰️ Integration

Synchronized with the [BrowserForge Engineering Studio](https://github.com/youh4ck3dme/browserforge-dashboard) to show real-time manufacturing data.
